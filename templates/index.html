Documentation

PyLang is a simple interpreted language written in python

General

You can use nextline or semicolon for a multiline program. Also there are built in functions and user defined functions to use. There is a input taking system but if you want to use them you have to download the source code and use them on your own machine. You can comment a line using '$' sign.

fun add(a, b) let sum = a + b; return sum; end let a = 10 let b = a + 5; println(a) print(add(5, b)) $print(add(5, b))

Loops

You may also use the good old for and while loops in PyLang. Basic increment of a for loop is equal to 1. You can customize increment using step keyword. You have to define the end point of a loop using to keyword(inclusive). While loop doesn't have any step property so you can change the variable as you want inside the loop.

for i = 1 to 6 then println("Hello World!") end for i = 1 to 11 step 2 then println("Hello World with step") end let i = 1 while i < 10 then println(i) let i = i + 2 end

Functions

You can define a function using fun keyword. You can define an one liner function using -> operator or a multiline function as you need. If you define a multiline function you have to put end keyword at the end of the function. You can assign a function to a variable and then use them with that variable. You can also define anonymous functions without a name.

fun add(a, b) let sum = a + b; return sum; end fun div(a, b) -> a/b; let a = fun () -> print( "Hello!" )

Built-ins

There are many built in functions which you can use. There are 2 function for displaying something on the screen.
1. println function is used for printing something with a newline on the screen.
2. print function print content without a newline.
3. len function return the length of a list.
4. int, float and str function convert a value to the specific type.
5. isNumber, isString, isList and isFunction is used to check the type of a value and they return true or false based on the result.

print("Hello World!") println("Hello World with a new line") println( len( [1, 2, 3, 4, 5] ) ) let a = int(1.43) let s = str(234) println( isString(s) )

Logical Operators

You can use logical comparison via and, or, and not. You can also use boolean true, false, null as variable initializer or comparison.

if true then println("This is true.") let a = 4 if a < 10 and a > 0 then println("a is between 1 and 9") end let flag = false if not flag then println("flag is false") else println("flag is true") end

Variable

You have to use the let keyword to "declare" and "re-assign" a variable.

let a = 10 let b = "Hello" let a = a + 1 let c = b + "World" let d = [1, 2, 3, 4, 5, 6]

Conditionals

You can use Conditionals with if, elif and else ladder. You have to put then keyword after every condition and end keyword after a ladder. You'll also be able to initialize your variables based on one liner conditionals.

let a = 10 if a < 10 then println("a is lesser than 10") elif a > 10 then println("a is greater than 10") else println("a is equal to 10") end let b = if 5 == 5 then 5 else 10; println(b)

List

You can asign a empty list or a initialized list to a variable.
1. To append a new value to the list use + operator.
2. To remove a element from any index of the list use - operator.
3. To concate 2 lists use * operator.
4. Use / operator to obtain a value from a certain index of a list.
5. You can get the length of a list by using len built-in function.

Remember all of these operations are immutable. You can do mutable operations on list using built-in functions.

let a = [] let a = a + 3 let a = a + 5 println(a) let b = [1, 2, 3] let c = a * b println(c) let c = c - 2 println(c) println(c / 2)

Mutable Built-ins

There are plenty of mutable list function which you can use. More will be added later.
push = (reference of a list, a value to push). It will append the value at the last index.
pop = (reference of a list, index number). It will return the removed element as well.
extend = (reference of first list, reference of second list). It will append the elements from second list to first list.
replace = (reference of a list, index number, a value). It will replace the value of that list at given index.

let list = [1, 2, 3, 4, 6] let a = [1, 2] push(list, 3) println(list) let element = pop(list, 2) println(element) println(list) extend(list, a) println(list) replace(list, 1, 100) println(list)
